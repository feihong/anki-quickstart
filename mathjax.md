## MathJax examples

This is inline MathJax: $\sqrt{x}$. This is a MathJax block:

$${\sum}_{i = 1}^{n} {i}^{3} = {\left(\frac{n \left(n + 1\right)}{2}\right)}^{2}$$

This is an alternate way to define a MathJax block:

```math
$${\sum}_{i = 1}^{n} {i}^{3} = {\left(\frac{n \left(n + 1\right)}{2}\right)}^{2}$$
```

You can put linebreaks in a MathJax block

$$
\displaylines{
x^2 + 6x + 11 = (x + 3)^2 + 2 \\
x^2 + 14x + 30 = (x+7)^2 -19 \\
x^2 - 2x + 7 = (x-1)^2 +6
}
$$
